CSS Text Formatting

A hierarchical list of tags to use for consistent formatting.

Tag Name Description
<font-family> Font Family A property used to specify the typeface in a rule set. Fonts must be available to the browser to display correctly, either on the computer or linked as a web font. If a font value is not available, browsers will display their default font. When using a multi-word font name, it is best to wrap them in quotes.
<font-size> Font Size Used to set text sizes. Font size values can be many different units or types such as pixels.
<font-style> Font Style The property determines the font style in which text will appear. For example, it can set the font style to italic.
<font-weight> Font Weight The property can be used to set the weight (boldness) of text. The provided value can be a keyword such as "bold" or "normal".
<line-height> Line Height The property declares the vertical spacing between lines of text. It accepts both unitless numbers as a ratio (eg. 2) and numbers specified by unit values (eg. 12px) but it does not accept negative numbers.
A unitless number is an absolute value that will compute the line height as a ratio to the font size and a unit number can be any valid CSS unit (eg. pixels, ems, percents, etc.)
<text-align> Text Align The property can be used to set the text alignment of inline contents. This property can be set to these values: left, center, or right.
<color> Color This property can change the foreground text color of an element. The value can be a valid color name supported in CSS like green. Also 3 digit or 6 digit color code like #ffffff can be used to set the color.
<background-color> Background Color The property controls the background color of elements.
<opacity> Opacity The property can be used to control the transparency of an element. The value of this property ranges from 0 (transparent) to 1 (opaque).
